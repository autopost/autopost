SeepCell

SeepCell is a puzzle game for Android.

Easy to play for all ages. Rotate the blocks, find the path from the top to the bottom of the board!

    Board is the main element of the game. The board consists of blocks, which consist of cells. The number of cells vertically and their number horizontally are equal. So the block is a square (the cell is also a square). Each cell might be filled (blue color means a cell is filled with water) or be empty (grey color).
    The player's goal is to rotate the blocks clockwise to ensure it is possible to traverse the board from any filled cell in the first row (top row of board cells) to any filled cell in the last (bottom) row through the filled cells of the board in a given time (1.5 minutes). The path can not pass through the cells located diagonally.
    This means schematically that water from the top penetrated to the lower level of the board. So the game name - Seep The Cell or short SeepCell!
